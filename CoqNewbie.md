Resources for Newbies
---------------------

Well, thinking in Coq is not hard. It just requires some time to learn syntax and basic tactics. After that you'll be able to create almost any possible proof. Of course, it's better to start with `Coq FAQ`_. The library also contains a lot of small proofs and can help a lot to get started: http://coq.inria.fr/library/.

But some questions are always interesting for any newbie:

ComparisonWithOtherSystems_

If you are searching some little task to contribute, you might want to look at

ProjectIdeas_

Novice coq users can submit questions to CoqNewbieQuestions_.
